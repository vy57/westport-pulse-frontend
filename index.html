Data Sources
  • Interest Rates: Fetched live from the Federal Reserve Economic Data (FRED) API (Series: MORTGAGE30US).
  • Demographics & Housing: Fetched from the US Census Bureau API (ACS 5-Year Estimates, 2022 dataset for ZIP 06880).
Core Formulas
  • Loan Amount: P = Home Price - (Home Price * Down Payment %)
  • Monthly Mortgage Payment (M): Calculated using the standard amortization formula:
    M = P [ r(1 + r)^n ] / [ (1 + r)^n - 1 ]
    Where "r" is the monthly interest rate (Annual Rate / 12) and "n" is the total number of payments (360 for a 30-year loan).
  • Top-Code Tax Bypass: The Census caps property tax reporting at $10,001. If the API detects this cap, it dynamically bypasses it by applying a standard 1.25% effective tax rate assumption against the median home value.
  • Affordability Score: True Monthly Housing Cost / Monthly Gross Income. A score above 0.36 triggers the "Cost Burdened" classification.
