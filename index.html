Behind the Code: Engineering My API

Rather than calculating projections off stationary tables, I built a serverless backend engine deployed on Cloudflare Workers. It acts as an integration gateway that queries live macro data directly from the Federal Reserve Bank of St. Louis (FRED) API for active 30-year fixed loan evaluations, while concurrently processing localized household and structural variables using the US Census Bureau API.

My codebase resolves census statistical caps by tracking tax distortions natively, generating dynamic debt-to-income models, and parsing immediate temporal snapshots to evaluate exactly how historical policy environments diverge from modern conditions.
